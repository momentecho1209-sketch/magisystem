ABOUT
// MAGI SYSTEMとは

MAGI SYSTEM は、複数のAIに同じ質問を投げかけ、それぞれの回答を比較できる実験的なAI回答比較サイトです。

3つのAI — ChatGPT（CASPER）、Gemini（BALTHASAR）、Claude（MELCHIOR） — がそれぞれ独立して質問を分析し、YES / NO の判定と理由を返します。最終的に多数決で「APPROVED（承認）」「DENIED（否決）」「SPLIT DECISION（意見分裂）」の結果が表示されます。

// 使い方

1. テキストボックスに質問を入力します（最大300文字）。

2.「EXECUTE」ボタンを押すと、3つのAIが同時に質問を分析します。

3. 各パネルに結果が表示されます。パネルをクリックすると、AIの判定理由を詳しく確認できます。

4. 結果は「SHARE ON X」ボタンでX（旧Twitter）に共有できます。

// 技術構成

フロントエンド: HTML / CSS / JavaScript（フレームワーク不使用）

バックエンド: Cloudflare Pages Functions（サーバーレス）

AI API: OpenRouter経由で複数のAIモデルに接続

セキュリティ: Cloudflare Turnstile（BOT対策）、KVベースのレート制限

// 利用制限

サーバー負荷軽減およびAPI利用コスト管理のため、以下の制限を設けています。

・5回質問すると60秒間のクールダウンが発生します。

・1回の質問は最大300文字までです。

・BOTによる自動アクセスはCloudflare Turnstileによりブロックされます。

// よくある質問（FAQ）

Q. 回答は正確ですか？
A. AIによる自動生成のため、正確性は保証されません。参考程度にご利用ください。

Q. 質問内容は保存されますか？
A. 当サイトでは質問内容を保存・蓄積していません。AIへの送信後、サーバー上にデータは残りません。

Q. 無料で利用できますか？
A. はい、完全無料でご利用いただけます。

Q. スマートフォンでも使えますか？
A. はい、レスポンシブデザインに対応しており、スマートフォンやタブレットからもご利用いただけます。
