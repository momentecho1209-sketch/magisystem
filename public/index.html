ABOUT THIS SITE
// このサイトについて

このサイトは、アニメ「エヴァンゲリオン」に登場する MAGIシステム にインスパイアされて制作された、MAGI風AI回答サイトです。

作品の雰囲気をモチーフにしたファンインスパイア作品であり、公式とは一切関係ありません。

本サイトでは、複数のAI（ChatGPT・Gemini・Claude）から回答を取得し、その結果を表示します。

// 使い方
  1. 質問を入力します
  2. 「EXECUTE」ボタンを押します
  3. AI（ChatGPT / Gemini / Claude）が回答を生成します
  4. それぞれのAIの回答を比較しながら閲覧できます
// 利用制限

サーバー負荷軽減のため、質問回数には制限があります。

5回質問すると60秒間のクールダウンが発生します。
